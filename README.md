### Paso a Paso
0. Crear Compartment FSDR en Stgo y Valparaíso
	Santiago
	Menu -> Identity & Security -> Compartmente -> New Compartment
	```
	CAMPO				VALOR
	==============================================
	Name		 		        FSDR_NOMBRE_stgo     
	Description 			  FSDR_NOMBRE
	Parent Compartment 	XXXX (root)
	```
 
	Valparaíso
 	Menu -> Identity & Security -> Compartmente -> New Compartment
	```
	CAMPO				VALOR
	==============================================
	Name		 		        FSDR_NOMBRE_valpo    
	Description 			  FSDR_NOMBRE
	Parent Compartment 	XXXX (root)
	```
